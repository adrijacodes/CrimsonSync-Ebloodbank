## ⚙️ Installation Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/e-blood-bank.git

   ```

2. Backend setup:
   ```bash
   cd server
   npm install
   ```
3. Frontend setup:
   ```bash
   cd client
   npm install
   ```
4. Run development server:

   ```bash
   npm run dev

   ```
